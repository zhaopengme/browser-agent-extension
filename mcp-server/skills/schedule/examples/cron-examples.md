# Cron 表达式示例

## 基本格式

```
┌───────────── 分钟 (0 - 59)
│ ┌───────────── 小时 (0 - 23)
│ │ ┌───────────── 日期 (1 - 31)
│ │ │ ┌───────────── 月份 (1 - 12)
│ │ │ │ ┌───────────── 星期 (0 - 6, 0=周日)
│ │ │ │ │
* * * * *
```

## 常用频率

### 按天

| 描述 | Cron 表达式 |
|------|------------|
| 每天凌晨 0 点 | `0 0 * * *` |
| 每天凌晨 1 点 | `0 1 * * *` |
| 每天上午 9 点 | `0 9 * * *` |
| 每天 12:30 | `30 12 * * *` |
| 每天晚上 11:59 | `59 23 * * *` |

### 按小时

| 描述 | Cron 表达式 |
|------|------------|
| 每小时整点 | `0 * * * *` |
| 每 30 分钟 | `*/30 * * * *` |
| 每 15 分钟 | `*/15 * * * *` |
| 每 10 分钟 | `*/10 * * * *` |
| 每 5 分钟 | `*/5 * * * *` |

### 按周

| 描述 | Cron 表达式 |
|------|------------|
| 每周一上午 9 点 | `0 9 * * 1` |
| 每周五下午 5 点 | `0 17 * * 5` |
| 每周日凌晨 2 点 | `0 2 * * 0` |
| 工作日每天上午 9 点 | `0 9 * * 1-5` |

### 按月

| 描述 | Cron 表达式 |
|------|------------|
| 每月 1 号凌晨 | `0 0 1 * *` |
| 每月 15 号中午 | `0 12 15 * *` |
| 每月最后一天 | `0 0 28-31 * *` |

### 复杂组合

| 描述 | Cron 表达式 |
|------|------------|
| 工作日每 2 小时 | `0 */2 * * 1-5` |
| 周末晚上 8 点 | `0 20 * * 0,6` |
| 每天 9am-5pm 每小时 | `0 9-17 * * *` |
| 1月和7月每天 | `0 0 * 1,7 *` |

## 特殊符号

| 符号 | 含义 | 示例 |
|------|------|------|
| `*` | 任意值 | `* * * * *` = 每分钟 |
| `,` | 列表 | `0 9,17 * * *` = 9am 和 5pm |
| `-` | 范围 | `0 9-17 * * *` = 9am 到 5pm |
| `/` | 步长 | `*/10 * * * *` = 每 10 分钟 |

## 注意事项

1. 时间是**服务器本地时区**
2. 星期 0 = 周日，6 = 周六
3. 日期和星期字段是**或**关系（满足任一即可）
4. ai-memory 使用 5 段式 cron（不含秒）
